Projects

Here are some of the projects that I have worked on.

Armored Acorns

Armored Acorns was a game that I worked on with a small startup. We were able to release it onto the NintendoWiiU Online store! We built the game using impactJs a rendering engine that makes use of chrome's canvas, and Nintendo Webframework. It was one of the first projects that I have worked on, and I had a great experience working on a team.

Website

leet-cord

This web application I created is heavily inpsired by Discord. It allows users to create Servers with channels to chat in real time! I built this application using Python / Flask / SQLAlchemy for the backend and for the frontend I used Javascript with React / Redux. I also implemented aws to allow users to upload custom server icons, or send images in chat rooms. Socket.io framework allows for messages to be updated in real time.

Mix Nebula

A music player web application that is inspired from the website SoundCloud. Users can browse the app and play tracks and albums from various artists. They can also sign up for an account and become an artist themselves! With AWS user can upload their own album cover art and sound tracks. The backend is built with Javascript / Express and Sequelize. The frontend is built with Javascript with React / Redux.

Live Link

InstaPostPix

A web application for sharing photos that is inspired by the popular app Instagram. Collaborated with 2 other developers, we created the backend using python / flask. The frontend was created with React / Redux. Users can post photos, comment on other posts and follow other users.
