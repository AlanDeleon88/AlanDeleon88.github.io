Armored Acorns

Armored Acorns is a game I developed with a small startup, which we successfully launched on the Nintendo WiiU online store. We created the game by using impactJs, a rendering engine leveraging Chrome's canvas, and a Nintendo Web framework. As one of my initial projects, working on a team to develop Armored Acorns was a fantastic experience.

leet-cord

This web application I developed draws significant inspiration from Discord. It enables users to create servers with channels for chatting. The backend was built using Python, Flask, and SQLAlchemy, while the front end was developed with JavaScript using React and Redux. Additionally, I integrated AWS to allow users to upload custom server icons or share images in chat rooms. I used the socket.io framework to facilitate real-time messaging.

mixNebula

Introducing a sleek and modern music player web application that's taking inspiration from the iconic SoundCloud platform. This app allows users to browse and enjoy tracks and albums from a diverse range of artists, and even create their own accounts to share their own musical creations. The backend technology stack includes Javascript/Express and Sequelize, while the frontend has been crafted using the latest Javascript frameworks: React/Redux.
